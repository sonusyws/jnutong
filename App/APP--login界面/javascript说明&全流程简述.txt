1. function checkInput()
   检查txtId、txtPassword、txtCaptcha内容是否都不为空
   如果是，把btnLogin颜色改成#609FEC
   否则，把btnLogin颜色改成#7AB6FF

2. function getLoginInfo()
   通过Ajax访问接口URL，获取登录所需的参数信息，设置验证码
   这个不会写就先不写函数体

3. function captchaRefresh()
   通过Ajax访问接口URL，刷新验证码
   这个不会写先不写函数体

4. function login()
   通过Ajax访问登录接口，若登录成功，跳转到另一页面
   这个不会写先不写函数体


登录页面(login.php)全流程：
1. 带id(GET参数)访问login.php，根据id参数从数据库中查找对应记录，利用记录中的数据设置txtId和txtPassword
2. 调用getLoginInfo()设置验证码，保存登录参数
3. 用户填写->点击登录后调用login()
